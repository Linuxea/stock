# 项目概览 (Project Overview)

这是一个用于展示中国A股和美股市场每日行情的静态 HTML 报告项目。该项目没有复杂的构建系统或依赖包管理，而是将每天的市场分析总结为独立的、结构化的 HTML 文件，并按日期归档。项目通过 CDN 引入 Tailwind CSS（用于样式）、Chart.js（用于数据可视化）和 Google Fonts（Noto Sans SC），具备暗黑模式切换、响应式设计等现代 Web 特性。

## 目录结构分析 (Directory Overview)

项目的核心是由一系列以日期（`YYYYMMDD` 格式）命名的目录组成，每个目录下存放当天的行情报告文件：

- `20250212/`, `20260211/`, `20260303/` 等：按日期分类的行情报告文件夹。
- `CLAUDE.md`：项目的架构说明及 AI 辅助开发指南，记录了颜色规范、新建报告的流程以及依赖说明。
- `.gitignore`：Git 忽略配置，排除了系统文件、编辑器缓存和临时文件等。

## 关键文件 (Key Files)

在每个日期的文件夹（如 `20260303/`）中，通常包含以下重要文件：

- `stock_report.html`：**A股市场行情报告**。包含上证、深证、创业板、科创50等核心指数的涨跌数据、两市成交量、板块追踪（如受地缘政治影响的油气、军工、黄金等避险板块）以及辩证维度的市场总结。
- `us_stock_report.html`：**美股市场行情报告**。结构与A股报告相似，用于记录和展示美股盘面表现。
- `*.png`：部分目录中包含的报告预览截图或图表输出文件。

## 使用说明 (Usage)

由于项目由纯静态文件组成，无需编译或构建。

**1. 本地预览报告**

进入对应的日期目录，启动本地 HTTP 服务器即可在浏览器中预览完整的样式与交互效果：
```bash
cd 20260303
python3 -m http.server 8080
# 然后在浏览器访问：http://localhost:8080/stock_report.html
```

**2. 创建新一期的行情报告**

当需要生成新一天的行情报告时，请遵循以下流程：
1. 建立以当天日期命名的新文件夹（例如：`YYYYMMDD/`）。
2. 从上一交易日的目录中复制模板（如 `stock_report.html`）到新目录。
3. 打开 HTML 文件进行手动更新：
   - 网页 `<title>` 以及 `<header>` 中的日期。
   - 大盘的核心指标（点位、涨跌幅数值等）。
   - 图表实例（Chart.js 配置里的 `data` 数组）。
   - 市场的宏观总结与板块解析文案。
4. 保存并在本地预览确认无误。

## 设计规范 (Design System)

- **颜色约定：** A股/中国市场适用“红涨绿跌”；美股市场则使用标准的“绿涨红跌”。
- **UI 组件：** 使用 `.gradient-bg`、`.glass-effect` 和 `.card-hover` 等 Tailwind/自定义 CSS 类来实现玻璃拟物化、卡片悬浮及渐变背景效果。
- **暗黑模式：** 支持通过页面右上角的按钮切换主题，图表和背景色会自动适配 `.dark` 类的样式覆盖，且用户偏好存储在 `localStorage` 中。
- **数据来源：** 报告数据主要引用自财联社、证券时报、新浪财经、同花顺、东方财富、Investing.com 等金融资讯平台。
